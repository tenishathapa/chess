Chess game supported by my own chess engine: [`endgame`](https://github.com/bikrantlabs/endgame)

## Project Setup

### Prerequisites

- Download `nodejs` from official website and install it.

### Clone the Repository

```bash
git clone git@github.com:bikrantlabs/chess.git
```

### Install Dependencies

Navigate inside the folder and type: `npm install`.

### Run the project

`npm run dev`

> Open `http://localhost:3000` to access the app.
